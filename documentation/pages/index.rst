Bool Tool Documentation
=======================

Welcome to the documentation of Bool Tool, a hard-surface modeling extension for Blender. Bool Tool is hosted on `Blender's official extensions platform <https://extensions.blender.org/add-ons/bool-tool/>`_ and can be installed from Blender's preferences directly (recommended). Source code of the add-on can be found on `Github <https://github.com/nickberckley/bool_tool>`_. This website only offers documentation for the latest version of the add-on.

❤️ You can support the long-term maintenance and growth of the project `by showing support! <https://buymeacoffee.com/nickberckley>`_
