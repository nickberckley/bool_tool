Bool Tool Documentation
=======================

Welcome to the documentation of Bool Tool, a hard-surface modeling extension for Blender. Bool Tool is hosted on `Blender's official extensions platform <https://extensions.blender.org/add-ons/bool-tool/>`_ and can be installed from Blender's preferences directly (recommended). Source code of the add-on (and this website) can be found on `Github <https://github.com/nickberckley/bool_tool>`_. This website only offers documentation for the latest version of the add-on.

If you have inquiries about the extension you can reach out on Github discussions. There you can also share content you made with the help of, or about Bool Tool. Anyone is welcomed to make video tutorials about the extension and share on platforms of their choosing. If you want to contribute to the documentation with screenshots, graphs, videos, or other content also reach out on Github discussions.

If you want to report a bug or request a feature `create a new issue on Github <https://github.com/nickberckley/bool_tool/issues>`_.

❤️ You can help the long-term maintenance and growth of the project `by showing support! <https://buymeacoffee.com/nickberckley>`_
